Ejemplos Avanzados

Descubre todas las capacidades de rojeru-toast con estos ejemplos avanzados

Toast con Acciones

Toast que incluye botones de acción personalizados.

const toast = rojeruToast.info(
    '¿Deseas guardar los cambios?',
    { duration: 0 }
);

// Agregar acciones después de mostrar
setTimeout(() => {
    const actions = document.createElement('div');
    actions.innerHTML = `
        <button onclick="guardar()">Guardar</button>
        <button onclick="descartar()">Descartar</button>
    `;
    toast.element.querySelector('.rojeru-toast-content')
          .appendChild(actions);
}, 100);

Progreso en Tiempo Real

Toast que muestra progreso de una operación larga.

const progressToast = rojeruToast.loading(
    'Procesando: 0%',
    { duration: 0 }
);

// Simular progreso
let progress = 0;
const interval = setInterval(() => {
    progress += 10;
    if (progress <= 100) {
        progressToast.update(
            `Procesando: ${progress}%`
        );
    }

    if (progress === 100) {
        clearInterval(interval);
        progressToast.complete('¡Completado!');
    }
}, 300);

Toasts en Cadena

Mostrar varios toasts en secuencia.

// Mostrar toasts en secuencia
rojeruToast.info('Paso 1: Iniciando proceso...');

setTimeout(() => {
    rojeruToast.success('Paso 2: Validando datos...');
}, 1000);

setTimeout(() => {
    rojeruToast.warning('Paso 3: Procesando información...');
}, 2000);

setTimeout(() => {
    rojeruToast.success('Paso 4: ¡Proceso completado!');
}, 3000);

Tiempo Extendido

Toast que cambia su duración dinámicamente.

const toast = rojeruToast.warning(
    'Sesión expirando en 5 minutos',
    { duration: 5000 }
);

// Extender tiempo si usuario está activo
document.addEventListener('mousemove', () => {
    if (toast.element && toast.element.parentNode) {
        // Reiniciar tiempo
        toast.element.querySelector('.rojeru-toast-progress')
              .style.transition = 'transform 5000ms linear';
        toast.element.querySelector('.rojeru-toast-progress')
              .style.transform = 'scaleX(1)';
    }
});

Toast con Sonido

Toast que reproduce sonido al mostrarse.

function playNotificationSound() {
    const audio = new Audio('notification.mp3');
    audio.play();
}

rojeruToast.success('¡Nuevo mensaje!', {
    onShow: () => {
        playNotificationSound();
    }
});
